*Libft_test, by Matthias Sars, <msars@student.42berlin.de>*

## Description
Simple test programs for the project 'Libft' from the 42 core curriculum (version 19.2). In the current version, the Norm and compilation behaviour are not checked. For part 1 and most of part 2 it gives passes and fails, where in part 1, the functions are checked against the libc functions. For the remainder of part 2 (the output functions) and all of part 3, it simply outputs stuff you can compare with what you expect.

## Instructions
My own Libft repo is contained as a submodule. To run it with other Libft repos, make sure your `libft.h` and `libft.a` are in the directory `libft` (lowercase 'l'). Compile with `make` and run with `./test_part1`, `./test_part2`, `./test_part2a` (the output functions like `ft_putstr_fd()` are in this separate program), or `./test_part3`.

*Libft_test, by Matthias Sars, <msars@student.42berlin.de>*

## Description
Simple test programs for the project 'Libft' from the 42 core curriculum (version 19.2). In the current version, the programs don't give pass or fail, but simply output stuff you can compare with what you expect. For many functions in part 1, the libc versions are called as well, so it is convenient to compare.

## Instructions
My own Libft repo is contained as a submodule. To run it with other Libft repos, make sure your `libft.h` and `libft.a` are in the directory `libft` (lowercase 'l'). Compile with `make` and run with `./test_part1`, `./test_part2`, or `./test_part3`.
